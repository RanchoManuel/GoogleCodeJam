Generalidades.

Este repositorio busca ser un lugar donde se acumule mi conocimiento en maratones de programación, específicamente en la "Google Code Jam". El código fuente consiste de un proyecto desarrollado en "eclipse"(IDE) y que por consiguiente sigue el formato de proyecto JAVA estándar de eclipse. Dentro se puede encontrar un conjunto de paquetes, donde cada uno se nombra igual que una competencia y agrupa los problemas de la misma.

Existe adicionalmente un paquete auxiliar("aux") donde se guarda el esqueleto básico que uso en los problemas.

El código que aquí se encuentra es libre(GPL v3).
